Denylist Globale vs Settings Locali

.claude/settings.json (repo)
{
  "permissions": {
    "allow": [
      "Bash(curl:*)",
      "Bash(rm:*)",
      "Bash(python3:*)"
    ]
  }
}
~/.claude/settings.json (globale)
{
  "permissions": {
    "deny": [
      "Bash(curl:*)",
      "Bash(rm:*)"
    ]
  }
}
curl / rm Allow locale + Deny globale
python3 Allow locale, no deny
