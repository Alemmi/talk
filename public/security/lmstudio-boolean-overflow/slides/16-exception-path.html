what happens
  • tokenizer.ggml.model = "t5" → the loader takes the t5 / UGM branch.
  • that branch copies our precompiled_charsmap blob onto the heap — before it checks the token list. The payload is planted.
  • why this array, not a spray KV the spray's s.* UINT8 arrays only blanket the fixed address 0x100000001; precompiled_charsmap is the one array the loader copies into its own fresh heap buffer (the assign above) — a single contiguous blob the unwinding lands in. The ROP has to live here.
  • we omit tokenizer.ggml.tokens, so the next check fails and the loader throws right after.
  • the throw unwinds the stack → ~llama_model runs — and walks our corrupted loras (next slide).
