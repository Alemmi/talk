what happens
  • unwinding runs ~llama_model, which iterates the loras set.
  • ① already overflowed past pimpl into loras — so every lora * is attacker-shaped, pointing into the spray ② at 0x100000001.
  • delete lora → ~llama_adapter_lora destroys its bufs vector.
  • each buffer's deleter calls ggml_backend_buffer_free — on pointers we control.
