void ggml_backend_buffer_free(ggml_backend_buffer_t buffer) {
  if (buffer == NULL) {                       // GADGET-1: first iter, blob[0] = 0 → returns silently
    return;
  }
  if (buffer->iface.free_buffer != NULL) {    // HIJACK: indirect call through attacker-controlled vtable
    buffer->iface.free_buffer(buffer);         //         resolved via the heap spray → stack-pivot gadget
  }
  delete buffer;
}

1st call Buffer pointer = NULL (from blob[0]) — the NULL check short-circuits, nothing happens.

2nd call Buffer = sprayed memory; iface.free_buffer resolves to a stack-pivot gadget. From here on, we own RIP.

No separate vtable corruption needed — we reuse an indirect call that already exists, reached during exception unwinding.
