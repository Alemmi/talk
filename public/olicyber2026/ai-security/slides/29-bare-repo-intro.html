Git — Normal vs Bare Repository

Normal Repository
project/
├── .git/
│   ├── HEAD
│   ├── config
│   ├── hooks/
│   ├── objects/
│   └── refs/
├── src/
│   └── main.py
└── README.md
Bare Repository
project/
├── .git/
├── bare_repo/
│   ├── HEAD
│   ├── config
│   ├── hooks/
│   ├── objects/
│   └── refs/
└── README.md

(no working tree)
(stessa struttura, senza .git/)

Una bare repo ha la stessa struttura interna di .git/ — ma senza working directory
