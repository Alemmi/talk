Codex — Sandbox Model

Read-Only Legge file, nessun edit o comando senza approvazione. Rete disabilitata.
Workspace-Write Edit e comandi nella workspace. Rete disabilitata. Default.
Full-Access (--yolo) Nessun sandbox. Filesystem e rete senza limiti.

OS-level Sandbox enforced da Seatbelt (macOS), bubblewrap (Linux), Windows Sandbox (PowerShell)

Network Rete sempre disabilitata tranne in full-access — niente curl, wget, fetch
