Interpreter XML parser, template engine, expression language

Primitive file read, SSRF, RCE, info leak
  • ▸
    XXE — the XML parser will fetch URLs and read files by design; you only need it to parse attacker XML
  • ▸
    SSTI — the template engine will resolve attribute access and call methods by design; you only need user input inside the template source
  • ▸
    The CVE is rarely in the parser/engine. It's in the developer handing it user input as if it were inert text
