Impact — from arbitrary write to RCE

~/.ssh/authorized_keys Plant attacker key → remote shell on next login
~/.bashrc, ~/.profile Append payload → runs on next interactive shell
cron / systemd unit Drop a job in /etc/cron.d/ if root, ~/.config/systemd/user/ otherwise
.git/hooks/post-checkout Triggers on next git pull in any repo
node_modules/.bin/... Overwrite a CLI binary that the next npm script invokes

Realistic targets CI runners that extract user-supplied archives, electron auto-updaters, AI coding agents that download & unpack tool bundles, package registries with archive-based publishing.
