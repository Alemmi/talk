Takeaways

  • ▸
    Async ≠ safe. Promise.all over user-controlled work items is concurrent code — analyse it like threads, not like a loop
  • ▸
    Count async hops, not microseconds. Inter-chain TOCTOU is decided by the structural difference in I/O hops, not by syscall durations
  • ▸
    A check on path X means nothing if another concurrent task is allowed to mutate path X. Either lock the path, hold an fd, or use O_NOFOLLOW
  • ▸
    Archive duplicates are weird. Both tar and zip allow same-named entries — treat them as a hostile signal, not a legitimate use case
  • ▸
    Audit the unused branches. The symlink branch had no preventWritingThroughSymlink — the protection only covered some code paths
