fs.writeFile internally = open(dest, 'w', mode) → write(fd, data) → close(fd)

What the kernel sees
open("dist/aaaaa",
     O_WRONLY | O_CREAT | O_TRUNC,
     0644)

dist/aaaaa is a symlink → /tmp/pwned

No O_NOFOLLOW → kernel resolves the link.
O_CREAT → if /tmp/pwned doesn't exist,
            create it.
O_TRUNC → if it does, truncate it.
